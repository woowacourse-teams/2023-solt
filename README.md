여행을 시작하는 설렘부터, 여행을 마무리하는 아쉬움까지도 트립드로우가 언제나 당신의 여행과 함께 합니다!   

트립드로우는 당신이 내딛는 소중한 걸음을 모두 그려드려요.     

언젠가 꼭 다시 오기로 한 맛집의 이름을 외우지 마세요.    

머리 위로 쏟아질듯한 별들을 언제 만났는지 신경 쓰지 마세요.    

트립드로우가 전부 빼놓지 않고 적어드릴게요.   
 
당신은 이 여행을 온전히 즐기기만 하세요.
